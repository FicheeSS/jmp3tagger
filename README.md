# JMP3Tagger
Un programme java qui permet de lire et modifier des tags sur des fichiers MP3 et de les renommer en accordance avec ces tags.

#Utilisation 
Pour utiliser le programme lancer *./out/artifacts/JMP3Tagger_jar/run_win.bat*
